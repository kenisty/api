post /v1/auth/register
A request to register a new user. (557dcb2ee3dfdae1e150def5ea1f831c)
Request for registering new users in. Returns a status code of 201 if the user was able to successfully register attached with an authentication token.

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Request body

UserRegisterReadV1 UserRegisterReadV1 (optional)
Body Parameter —

Return type

UserRegisterWriteV1

Example data

Content-Type: application/json
{
  "data" : {
    "firstname" : "firstname",
    "email" : "email",
    "token" : "token",
    "lastname" : "lastname"
  },
  "message" : "message"
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

201

Successfully registered a new user. UserRegisterWriteV1

Failed validation.
